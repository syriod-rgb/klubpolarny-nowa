# Polski Klub Polarny — nowoczesna strona

Nowoczesna, jednostronicowa witryna Polskiego Klubu Polarnego, zbudowana na
oryginalnych treściach i zdjęciach klubu. Statyczny HTML/CSS/JS (bez frameworków,
bez zależności build) — gotowy pod GitHub Pages.

Inspiracja wizualna: immersyjne, fotograficzne strony arktyczne (np. eastgreenland.com)
— pełnoekranowe kadry, edytorska typografia, narracja sekcjami. Charakter własny:
archiwalne szklane negatywy z **Polskiej Wyprawy na Grenlandię (1937)** jako motyw
przewodni dziedzictwa.

## Struktura

- `index.html` — cała strona (sekcje: Hero, O Klubie, Działalność, Dziedzictwo 1937,
  Aktualności, Galeria, Dołącz, Kontakt)
- `assets/css/styles.css` — style
- `assets/js/app.js` — interakcje (sticky header, menu mobilne, animacje przewijania, lightbox)
- `assets/img/` — wyselekcjonowane zdjęcia z archiwum klubu

## Treści i archiwum

Strona prezentuje wybrane treści; pełne archiwum (wszystkie artykuły, galerie, dokumenty)
linkowane jest do kompletnej kopii serwisu:
**https://syriod-rgb.github.io/klubpolarny-pl/**

## Uruchomienie

- Lokalnie: otwórz `index.html` w przeglądarce.
- GitHub Pages: *Settings → Pages* → gałąź `main`, katalog `/ (root)`.
  Plik `.nojekyll` wyłącza przetwarzanie Jekyll.

## Prawa

Treści, fotografie i materiały należą do **Polskiego Klubu Polarnego**.
Fotografie archiwalne pochodzą z dokumentacji Polskiej Wyprawy na Grenlandię (1937).
